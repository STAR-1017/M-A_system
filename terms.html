利用規約

第1条　適用

  1. 本利用規約（以下、「本規約」とする。）は、ラッコ株式会社（以下、「当社」とする。）が提供するウェブサイトの売買に関するサービスを提供するサイト「ラッコM&A」（以下、「当サイト」とする。）の利用にあたり、利用者が遵守すべき事項、および、当社と利用者の間の権利義務について定めています。
  2. 本規約とは別に、当社が当サイト上に記載する規約は、本規約の一部を構成します。なお、規約の内容が異なる、あるいは矛盾する場合は、本規約を優先して適用するものとします。
  3. 利用者は本サービスの利用にあたり、本規約を遵守しなければなりません。本サービスに会員登録を申請し、会員として登録された場合、利用者は本規約に同意したものとします。

第2条　定義

  1. 本規約において、「本サービス」とは、当社がウェブサイトの売却ならびに購入を希望する者に対し、当サイト、あるいは当社を通じて、売買に関する情報とそれに関する業務を提供するサービスを意味します。なお、本サービスはウェブサイトの売買に関する仲介を行うものではありません。
  2. 本規約において、「売主」とは、当サイト、あるいは当社を通じて、所有するウェブサイトの販売を希望する者を意味します。
  3. 本規約において、「買主」とは、当サイト、あるいは当社を通じて、ウェブサイトの購入を希望する者を意味します。
  4. 本規約において、「代理人」とは、当サイト、あるいは当社を通じて、売主あるいは買主を代理してウェブサイトの売却・購入を希望する者を意味します。
  5. 本規約において「知的財産権」とは、著作権、特許権、実用新案権、商標権、意匠権 その他の知的財産権、およびそれらの権利の登録等を出願する権利を意味します。
  6. 本規約において「エスクローサービス」とは、サイトの売買における決済方法として、売主と当社との間で、売買代金の受領口座として当社の預金口座を指定いただき、当社が買主から売買代金の受領を受けた後、買主において第11条1cの規定に従った目的物の検収にいたったとき、売主に当該売買代金を交付することを合意することで、売買代金決済の確実性を高めるサービスです。
  7. 本規約、本サービスにて「サイト自動査定」とは、当社の独自の査定項目により、売買価格の目安となる金額を表示するサービスです。

第3条　利用

  1. 当サイトの利用を希望する者は、当社の定める一定の情報を、当社が定める方法で当社に提供することにより、当サイトを利用することができます。
  2. 満20歳未満の者は当サイトを利用することはできません。
  3. 正当な権限のない代理人による当サイトの利用は認められません。当社は、正当な代理権限の有無を確認するために委任状の提示等を求めることがあります。また、利用者及び代理人は利用の申請にあたり、真実、正確かつ最新の情報を当社に提供しなければなりません。代理人は売主あるいは買主と委任契約を締結している必要があり、また本規約において、売主の代理人は売主と、買主の代理人は買主と等しい権利義務を利用者として負います。
  4. 当社は、当社の基準にしたがって当サイトの利用の可否を判断します。
  5. 当社は、前項の判断、サービス利用の適正を確認するために、商品となるウェブサイトの情報、売買契約の内容に関する情報を取得することができます

    • 売主は、買主からの連絡に対し、できるだけ速やかに検討を行い、遅滞なく、商談の受諾、あるいは商談の拒否を通知するものとします。
    • 本サービスの利用における決済は、当社が提供するエスクローサービスを利用することとします。
    • 本サービスの利用の結果、口頭、書面を問わず、契約を締結した場合は、契約当時者は直ちに、目的物の名称、契約代金、譲渡物、その他当社の定める事項を当社の定める様式にしたがって当社に報告するものとします。
    • 売主は、目的物の譲渡が決定した後は譲渡完了まで誠実に譲渡手続を進めるものとします。利用者が正当な理由なく譲渡手続を遅延する場合、当社は、相当の期間を定めて手続の進行を催告し、その期間内に進行がないときは、譲渡手続を打ち切ることができるものとします。このとき、譲渡手続を進めなかった当事者は、当社に対して、契約打ち切りまでに生じた損害を賠償するものとします。賠償額は、譲渡が完了していれば当社が得ていた利用料金と同額とします。

    • 取引成立後には、当社の取扱実績として取引サイトが特定されないようにジャンル、金額等の掲載を継続いたします。
  3. 利用者は、本サービスの利用にあたり、以下の各号のいずれかに該当する行為をしてはなりません。
    • 当社、本サービスの他の利用者および第三者の権利、利益等を侵害する行為

第7条　売買の対象として取り扱わないウェブサイトについて

第8条　サイト自動査定

  2. 当社が取得したデータ、ならびにサイト自動査定の結果は、当社において保存し、査定精度向上のために利用します。
  3. 当社が取得したデータ、ならびにサイト自動査定の結果は、適正に取り扱い、第三者に開示、提供はしません。

  1. 当社は、売主が売却するサイトについての客観的信用性を高める目的で、googleアカウント連携に同意をした場合、当社は本条第4項に記載するアクセスデータを受領します。
  2. 当社は、買主への信頼性を高めるために、前項により取得したデータを利用者に開示します。当社は適宜のグラフ化等を行いますが、その元となるアクセスデータの改変を行いません。
  3. 当社は当該サイトの売却サイト登録が終了した場合には、取得したグーグルアカウントのアクセストークンを破棄します。
  4. 受領するGoogle Analyticsアクセスデータ

第10条　代金の決済・エスクローサービス

  1. 当社が提供する「エスクローサービス」とは、サイトの売買における決済方法として、当社が買主から売買代金の受領を受けた後、買主において目的物の検収にいたった後に、当社を介して売主に当該売買代金を送金する決済方法であり、売買代金決済の確実性、買主の利便性を高めるサービスです。なお、当社が受領した売買代金はエスクロー受領専用口座にて管理します。
  2. 利用者は、当社が提供するエスクローサービスを利用するものとします。なお、利用料金にエスクローサービスの費用は含まれます。
  3. エスクローサービスの利用は、以下の手順、あるいは当サイトにおけるエスクローサービスの説明に従い行なうものとします。
    • 売主、買主間で売買条件を定め、売買契約を締結します。
    • 買主は、当社が管理する預金口座に、売買代金および当社が別に定める利用料金を振り込みます。
    • 売主は、買主に対し、譲渡対象物を移転します。移転とは、ドメインの移転手続、レンタルサーバーの契約上の地位の移転、各IDおよびパスワードの通知等を指します。
    • 買主は、当事者間での契約に定める期間内に目的物の検収を行い、その結果を当社ならびに売主に通知します。なお、当社指定または個別の譲渡契約に定めた期間内に検収結果の通知がない場合には、目的物の検収が完了したものとみなします（検収不具合の場合はf）。
    • 当社は、dの買主による検収後、売買代金を売主が指定した口座に送金します。
    • 検収の結果、不合格とするのに正当な理由がある場合、買主は売主に30日間を上限とした任意の期間を定め、サイトの修正、情報の追加提供等の改善策を要請し、売主がこれに応じたときは、再度検収を行うものとし、以後、dに戻ります。
  4. 仮に、売主が検査期間内に改善をせず、かつ、当社に改善しないことの正当な理由を通知しない場合、当社は、譲渡契約が解除されたものとみなして売買代金および利用料金を買主に返還することができるものとします。
